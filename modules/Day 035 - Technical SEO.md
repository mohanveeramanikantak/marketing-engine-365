# Day 035 - Technical SEO

## Introduction

Technical SEO is an important part of Search Engine Optimization (SEO) that focuses on improving the technical structure and performance of a website. While On-Page SEO focuses on content optimization and Off-Page SEO focuses on backlinks and authority, Technical SEO ensures that search engines can properly crawl, index, and understand website pages.

Search engines like Google use bots or crawlers to scan websites. If a website has technical problems such as slow speed, broken links, poor mobile design, or indexing issues, search engines may struggle to rank the website properly.

Technical SEO helps improve:

* Website performance
* Crawlability
* Indexing
* Security
* User experience
* Mobile responsiveness

In digital marketing, Technical SEO is essential because even high-quality content may fail to rank if the website structure is technically weak.

In college marketing, educational institutions use Technical SEO to optimize admission websites, improve mobile experiences, secure student portals, and ensure better search visibility.

---

# Definition of Technical SEO

Technical SEO can be defined as:

> “The process of optimizing a website’s technical infrastructure to help search engines crawl, index, and rank pages effectively.”

It focuses on:

* Website accessibility
* Performance optimization
* Search engine crawling
* Mobile usability
* Security

---

Technical SEO is important because it supports website visibility and user experience.

# Crawling and Indexing

Search engines use bots called crawlers to discover website pages.

---

# Crawling

Crawling is the process where search engine bots scan website pages and links.

## Example:

Googlebot visiting a college admission website.

---

# Indexing

Indexing is the process of storing webpage information in search engine databases.

Only indexed pages can appear in search results.

---

# XML Sitemap

An XML sitemap is a file that lists important website pages.

## Purpose:

* Helps search engines discover pages
* Improves crawling efficiency
* Supports website indexing

### Example:

> website.com/sitemap.xml

Sitemaps are important for large websites.

---

# Robots.txt

The robots.txt file tells search engine bots which pages they can or cannot access.

## Example:

```txt
User-agent: *
Disallow: /admin/
```

This prevents search engines from crawling private pages.
